# Expense Tracker (Трекер расходов)

**Автор:** Лындин

## Описание программы
Expense Tracker — это GUI-приложение для отслеживания личных расходов, разработанное на Python с использованием библиотеки Tkinter. Программа позволяет добавлять, удалять и фильтровать расходы, автоматически подсчитывает общую сумму за выбранный период, а также сохраняет и загружает данные в формате JSON.

## Основные возможности
- Добавление расходов с указанием суммы, категории и даты
- Просмотр всех записей в удобной таблице
- Фильтрация по категории и диапазону дат
- Автоматический подсчёт общей суммы отфильтрованных расходов
- Сохранение и загрузка данных через JSON
- Валидация вводимых данных (сумма — положительное число, дата — формат ДД.ММ.ГГГГ)
- Удаление отдельных записей

## Требования
- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)

## Как использовать

### Запуск программы
1. Убедитесь, что Python установлен на вашем компьютере.
2. Скачайте файл `expense_tracker.py`.
3. Запустите программу командой:
   ```bash
   python expense_tracker.py
Добавление расхода
Введите сумму в поле Сумма (например, 1500).

Выберите категорию из выпадающего списка (или введите свою, если разрешено).

Введите дату в формате ДД.ММ.ГГГГ (например, 15.03.2024).

Нажмите кнопку Добавить расход.

Фильтрация
По категории: выберите нужную категорию в блоке «Фильтрация» и нажмите «Применить фильтр».

По дате: укажите начальную и/или конечную дату (формат ДД.ММ.ГГГГ) и нажмите «Применить фильтр».

Сброс фильтров: нажмите «Сбросить фильтр», чтобы увидеть все записи.

Удаление расхода
Выделите запись в таблице и нажмите кнопку Удалить выбранный расход.

Сохранение данных
Данные автоматически сохраняются в файл expenses.json в той же папке при каждом добавлении или удалении.

Примеры использования
Тест 1 – Корректное добавление
Сумма: 2500

Категория: Транспорт

Дата: 01.04.2024

Результат: Запись появляется в таблице, общая сумма увеличивается на 2500.

Тест 2 – Ошибка валидации суммы
Сумма: -100

Категория: Еда

Дата: 05.05.2024

Результат: Сообщение об ошибке «Сумма должна быть положительным числом!».

Тест 3 – Ошибка формата даты
Сумма: 500

Категория: Развлечения

Дата: 31.02.2024

Результат: Сообщение об ошибке «Дата должна быть в формате ДД.ММ.ГГГГ».

Тест 4 – Фильтрация по категории
Выбрать категорию «Еда» в фильтре, нажать «Применить фильтр».

Результат: Отображаются только расходы с категорией «Еда». Общая сумма пересчитывается для отфильтрованных записей.

Тест 5 – Фильтрация по диапазону дат
Ввести «Дата с: 01.03.2024», «по: 31.03.2024», нажать «Применить фильтр».

Результат: Отображаются только расходы за март 2024 года.

Тест 6 – Пустые поля
Оставить любое поле пустым и нажать «Добавить расход».

Результат: Сообщение «Все поля должны быть заполнены!».
