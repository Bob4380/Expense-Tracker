# Expense Tracker (Трекер расходов)

**Автор:** Лындин

## Описание программы

Expense Tracker - это GUI-приложение для отслеживания личных расходов, разработанное на Python с использованием библиотеки Tkinter. Программа позволяет вести учет расходов, фильтровать их по категориям и датам, а также просматривать общую сумму расходов за выбранный период.

## Основные возможности

- Добавление расходов с указанием суммы, категории и даты
- Просмотр всех расходов в удобной таблице
- Фильтрация расходов по категории
- Фильтрация расходов по диапазону дат
- Автоматический подсчет общей суммы расходов
- Сохранение и загрузка данных в формате JSON
- Валидация вводимых данных
- Удаление отдельных записей

## Требования

- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)

## Как использовать

### Запуск программы

1. Убедитесь, что Python установлен на вашем компьютере
2. Скачайте файл `expense_tracker.py`
3. Запустите программу командой: python expense_tracker.py

### Добавление расхода

1. Введите сумму в поле "Сумма" (только положительные числа)
2. Выберите категорию из выпадающего списка или введите свою
3. Введите дату в формате ДД.ММ.ГГГГ (например, 01.01.2024)
4. Нажмите кнопку "Добавить расход"

### Фильтрация расходов

**По категории:**
- Выберите категорию из выпадающего списка в секции "Фильтрация"
- Нажмите "Применить фильтр"

**По дате:**
- Введите начальную и/или конечную дату в полях "Дата с" и "по"
- Нажмите "Применить фильтр"

**Сброс фильтров:**
- Нажмите кнопку "Сбросить фильтр" для отображения всех расходов

### Удаление расхода

1. Выберите запись в таблице
2. Нажмите кнопку "Удалить выбранный расход"

### Сохранение данных

Данные автоматически сохраняются в файл `expenses.json` при каждом добавлении или удалении расхода.

## Примеры использования

### Тест 1: Добавление расхода
Сумма: 1500
Категория: Еда
Дата: 15.03.2024
Ожидаемый результат: Запись добавлена в таблицу, общая сумма обновлена

### Тест 2: Проверка валидации
Сумма: -100
Ожидаемый результат: Сообщение об ошибке "Сумма должна быть положительным числом!"

### Тест 3: Фильтрация по категории
Выбрать категорию "Транспорт" в фильтре
Ожидаемый результат: Отображаются только расходы категории "Транспорт"

### Тест 4: Фильтрация по дате
Дата с: 01.03.2024
Дата по: 31.03.2024
Ожидаемый результат: Отображаются расходы только за март 2024 года

### Тест 5: Некорректная дата
Дата: 31.02.2024
Ожидаемый результат: Сообщение об ошибке "Дата должна быть в формате ДД.ММ.ГГГГ"
