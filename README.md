# Expense Tracker (Трекер расходов)

## Автор
Лындин Никита Витальевич

## Описание программы
Expense Tracker - это GUI-приложение для отслеживания личных расходов. Программа позволяет пользователям записывать свои расходы, просматривать их в удобной таблице, фильтровать по категориям и датам, а также просматривать итоговую сумму расходов за выбранный период.

### Основные функции:
- Добавление расходов с указанием суммы, категории и даты
- Просмотр всех расходов в табличном виде
- Фильтрация расходов по категории
- Фильтрация расходов по диапазону дат
- Подсчет итоговой суммы расходов за выбранный период
- Сохранение данных в JSON формате
- Автоматическая загрузка данных при запуске

## Как использовать

### Запуск программы
1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Скачайте файлы программы
3. Запустите файл `expense_tracker.py`:
```bash
python expense_tracker.py
Добавление расхода
Введите сумму расхода в поле "Сумма"

Выберите категорию из выпадающего списка (еда, транспорт, развлечения, здоровье, одежда, другое)

Введите дату в формате ДД.ММ.ГГГГ (по умолчанию текущая дата)

Нажмите кнопку "Добавить расход"

Фильтрация расходов
По категории: выберите нужную категорию из выпадающего списка "Категория" в секции "Фильтрация"

По дате: введите диапазон дат в поля "С даты" и "По дату" в формате ДД.ММ.ГГГГ

Нажмите "Применить фильтр" для отображения отфильтрованных данных

Используйте кнопку "Сбросить" для отображения всех расходов

Просмотр итогов
Итоговая сумма расходов за отфильтрованный период отображается внизу окна программы.

Примеры использования
Пример 1: Базовое использование
Добавьте расход: 500 руб., "еда", 15.03.2024

Добавьте расход: 100 руб., "транспорт", 16.03.2024

Добавьте расход: 1000 руб., "развлечения", 20.03.2024

Программа отобразит все 3 расхода с итоговой суммой 1600 руб.

Пример 2: Фильтрация по категории
Выберите категорию "еда" в фильтре

Нажмите "Применить фильтр"

Отобразится только расход на 500 руб.

Итоговая сумма: 500 руб.

Пример 3: Фильтрация по датам
Введите "С даты: 15.03.2024"

Введите "По дату: 16.03.2024"

Нажмите "Применить фильтр"

Отобразятся расходы за 15-16 марта: 500 + 100 = 600 руб.

Тестирование валидации
Попробуйте ввести отрицательную сумму (например, -100) - будет показана ошибка

Попробуйте ввести неверную дату (например, 31.13.2024) - будет показана ошибка

Попробуйте оставить поля пустыми - будет показана ошибка

Техническая информация
Данные сохраняются в файл expenses.json в формате JSON

Приложение использует стандартную библиотеку tkinter для GUI

При первом запуске создается пустой список расходов
